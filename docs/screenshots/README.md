# Screenshots

Drop screenshots / GIFs of the add-in here, then reference them from the main
[`README.md`](../../README.md) (uncomment the image lines in its **Screenshots**
section).

## How to capture (Windows)
1. Open Word with the **Insert Formula** pane open.
2. Press **Win + Shift + S**, drag to select the pane (and a bit of the document
   showing the inserted result), and the snip is copied to the clipboard.
3. Paste into Paint (or Snip & Sketch) and **Save as PNG** into this folder.
   *(macOS: **⌘ + Shift + 4** to snip to a file on the Desktop, then move it here.)*
