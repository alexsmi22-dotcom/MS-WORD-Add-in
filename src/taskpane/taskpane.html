Examples & syntax
  • H2O → H2O
  • Ca(OH)2 → Ca(OH)2
  • SO4^2- → SO42−
  • Na+ → Na+
  • x^2 + y^2 → x2 + y2 (math mode)
  • a_n, x^{n+1}, sqrt(x), pi (math mode)

2D structures (Chemical mode) work for common names (e.g. aspirin, water), known formulas (C6H6), or any SMILES string (CC(=O)O). For stereochemistry, enter an isomeric SMILES (C[C@@H](N)C(=O)O) — it's drawn with wedges.

Build mode renders a structure from a typed atom/bond list or a pasted molfile. Atom/bond example:
atoms: C O O
bonds: 1=2 1=3 → CO2. Bonds: - single, = double, # triple, ~ undefined. Atoms can carry a charge (N+, O-). Hydrogens are added automatically. Stereo bonds: > wedge, < hash. For generic (Markush) structures: [C,N] variable atom, X halogen, A any atom, Q heteroatom, R1 R-group, ~ any-bond.

Drafting aid only — verify all structures and formulas before filing. Runs entirely on your machine; no document content is sent anywhere.
