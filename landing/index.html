Formula Inserter

A free Microsoft Word add-in for STEM authoring — formulas, equations, structures, units, plots, finance, and more. 14 modes, 100% offline.

🧪 Chemistry ∑ Math / LaTeX 📏 Units 📈 Plotting 💵 Finance 🧬 DNA / Sequences ⚗️ Reactions
View on GitHub Install (Windows & Mac)

What it does

🧪 ChemicalFormatted formulas, 2D structures from name/SMILES, name lookup
∑ MathNative Word equations, LaTeX import/export, aligned equations
📏 UnitsSI typesetting + conversion (incl. km/h → m/s)
📈 PlotOffline function & data charts with legends
💵 FinanceTVM, loan, NPV/IRR, Black–Scholes, bond calculators
🔬 BuildStructures from atom/bond lists; Markush genus
💻 CodeAlgorithm & code listings
🧬 SequenceWIPO ST.26 biological sequence listings
🌿 BotanicalPlant-nomenclature typesetting
🔢 Numerals / 🔖 RefsReference numerals, captions & cross-references
🧬 DNAReverse complement, translation, ORFs, primer Tm, restriction sites
✅ AuditWhole-document consistency check
Private by design. Everything runs on your machine — nothing you type ever leaves your computer. Works on Word for Windows & macOS (Microsoft 365 / 2019+).

Get it

Download the install pack for your OS from the install folder, unzip, and run the installer (full steps in the install guide). Then open Word → Insert → Add-ins → Formula Inserter.
