Formula Inserter

Type a formula, choose a mode, and insert it at the cursor.

Preview
2D structure

Examples & syntax
  • H2O → H2O
  • Ca(OH)2 → Ca(OH)2
  • SO4^2- → SO42−
  • Na+ → Na+
  • x^2 + y^2 → x2 + y2 (math mode)
  • a_n, x^{n+1}, sqrt(x), pi (math mode)

2D structures work for common names (e.g. aspirin, water), known formulas (C6H6), or any SMILES string (CC(=O)O).
