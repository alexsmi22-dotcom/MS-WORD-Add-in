JurisLab

A free Microsoft Word add-in fusing STEM authoring and patent & legal drafting — formulas, equations, structures, units, plots, finance modeling, Bluebook citations & Table of Authorities, and more. 16 tools, 100% offline.

🧪 Chemistry ∑ Math / LaTeX 📏 Units 📈 Plotting 📊 Table → Chart 💵 Finance 🧬 DNA / Sequences ⚗️ Reactions ⚖️ Citations & Authorities
View on GitHub Install (Windows & Mac)

New in this release

⚖️ Table of Authorities NewScan a brief and build a grouped, alphabetized authorities table — a static list, or live Word fields with real page numbers.
🧬 Sequence CDS annotation NewAnnotate CDS / gene features on ST.26 listings — a CDS auto-generates /translation from the coding region.
💵 Finance modeling NewDCF with terminal value, XIRR, bond YTM / duration / convexity, option Greeks & implied volatility, amortization & depreciation schedules.
🧪 Chemical-formula validator NewChecks every element symbol against the real 118-element periodic table and reports molecular weight and net charge.
📊 Stacked charts NewStacked column, bar & area (plus scatter) — with matching PowerPoint export.
⚖️ Citation styles & parallel cites NewPractitioner / academic typefaces, Table T6 & T10 case-name abbreviation, id./supra helpers, and parallel citations.

Private by design. Everything runs on your machine — nothing you type ever leaves your computer. Works on Word for Windows & macOS (Microsoft 365 / 2019+).

Get it

Download the install pack for your OS from the install folder, unzip, and run the installer (full steps in the install guide). Then open Word → Insert → Add-ins → JurisLab.
